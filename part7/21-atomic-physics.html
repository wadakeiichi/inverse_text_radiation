第21章　線スペクトルから原子物理へ

この章の主題：第19章で観測される線スペクトルを「ファクト」として確認し、第20章で線形成の枠組み（Einstein 係数と源泉関数）を整えた。本章では中心地図 \alpha_\nu = (\pi e^2/m_e c) f_{lu} n_l \phi(\nu-\nu_0) の f_{lu}（振動子強度） と \nu_0（線振動数） の起源を、量子力学から逆引きする。これは連続スペクトルにおける「第IV部の量子論」に対応する、線スペクトル側の 量子力学への逆引き である。

この章の中心地図

水素線スペクトルの地図 ― 二層構造

第一層：イコニック・アンカー（リュードベリ公式）

\frac{1}{\lambda} = R_\mathrm{H} \left( \frac{1}{n_1^2} - \frac{1}{n_2^2} \right) \tag{24.1}

線スペクトルの「骨格」を一目で示す。R_\mathrm{H} は基本定数の組合せ、1/n^2 は束縛状態のエネルギー、離散性が「線が存在する理由」を語る。

第二層：構造的マップ（Hα を標準例とした線吸収係数）

\alpha_\nu = \frac{\pi e^2}{m_e c} \, f_{lu} \, n_l \, \phi(\nu - \nu_0) \tag{24.2}

因子 物理的意味 対応する章
\alpha_\nu 観測量（線吸収係数） 第20章
\pi e^2 / m_e c 古典極限の線強度 第21章＋第VIII部
f_{lu} 振動子強度 第21章
n_l 下準位の占有数 第20章＋Part III
\phi(\nu - \nu_0) 線形（自然・Doppler・Voigt） 第22章
\nu_0 線位置（リュードベリ） 第21章

地図の使い方：第VII部の各章はこの式の異なる因子を解き明かす。プランク関数地図と並べて読めば、本書を貫く二つの逆引きが同時に把握できる。第VIII部の QED で、両者は前因子 2h\nu^3/c^2 を介して統一される（式 28.4 参照）。

ノート

方針：本章では水素原子を中心に置き、線振動数 \nu_0 と振動子強度 f_{lu} を量子力学から逆引きする。多電子原子と分子は概要に留め、第VIII部（QED）で電磁場と物質場の量子化として統一する。

この章で答える問い

  • 水素原子のスペクトル（Lyman・Balmer・Paschen 系列）は、なぜ Rydberg 公式で書けるのか
  • 振動子強度 f_{lu} は何を、どの近似で表しているのか
  • 選択則と禁制線の物理的根拠は何か
  • 水素原子の枠組みは、多電子原子・分子にどう拡張されるのか

到達目標

この章を読み終えると、読者は次のことができるようになる：

  • Rydberg 公式と水素原子のエネルギー準位を量子力学から再導出できる
  • 振動子強度 f_{lu} の物理的意味と Schrödinger 方程式との関係を述べられる
  • 選択則の起源と、それを破る現象（禁制線）の役割を説明できる

21.1 Rydberg 公式 ― 水素線の振動数の起源

[本文目安：B1-B2]

第19章 §19.1 で見た太陽の Fraunhofer C 線（\lambda = 656 nm, H\alpha）と、それに続く水素線系列の振動数は、すべて Rydberg 公式 で書ける：

\frac{1}{\lambda} = R_H \left(\frac{1}{n_l^2} - \frac{1}{n_u^2}\right) \tag{24.3}

ここで R_H \simeq 1.097 \times 10^7 m^{-1} は Rydberg 定数、n_l, n_u は主量子数。

  • Lyman 系列（n_l = 1）：紫外、\lambda < 122 nm
  • Balmer 系列（n_l = 2）：可視、H\alpha = 656 nm, H\beta = 486 nm, …
  • Paschen 系列（n_l = 3）：近赤外
  • Brackett、Pfund 系列（n_l = 4, 5）：赤外〜遠赤外

これは経験式として 1888 年に発見された。なぜこの形になるか を逆引きするのが本節以降の主題。

ノート

対応（観測）：観測可能なすべての水素線振動数が 二つの整数 n_l, n_u で記述できる という事実は、本書の主題「観測スペクトルから量子物理を逆引きする」の典型例。1888 年時点でこの規則性が認識されたが、それが何を意味するかは、Bohr 模型（1913）まで待たねばならなかった

21.2 Bohr 模型 ― 線振動数の最初の説明

[本文目安：B1-B2]

Bohr（1913）は、Rutherford の原子模型に 角運動量量子化 を導入した：

L = n\hbar, \quad n = 1, 2, 3, \ldots \tag{24.4}

これと古典電磁気の Coulomb 力 + Newton 運動方程式を組み合わせると、エネルギー準位

E_n = -\frac{1}{2} \frac{m_e e^4}{(4\pi\epsilon_0)^2 \hbar^2} \frac{1}{n^2} = -\frac{13.6\,\text{eV}}{n^2} \tag{24.5}

が出る。線振動数は h\nu = E_u - E_l から

\frac{1}{\lambda} = \frac{m_e e^4}{(4\pi\epsilon_0)^2 \cdot 4\pi \hbar^3 c} \left(\frac{1}{n_l^2} - \frac{1}{n_u^2}\right) = R_H \left(\frac{1}{n_l^2} - \frac{1}{n_u^2}\right) \tag{24.6}

これで Rydberg 公式が量子物理から 逆引き された。

注意（観測）：Bohr 模型は 線振動数は完璧に予言する が、振動子強度 f_{lu} は予言できない。なぜなら、模型は電子の位置を許される軌道の上に「正確に」置く描像で、波動関数を持たない。f_{lu} は次節の Schrödinger 方程式と波動関数によって初めて計算できる

21.3 Schrödinger 方程式と水素原子の波動関数

[本文目安：B2-B3]

電子の波動的性質を取り入れた Schrödinger 方程式（1926）を Coulomb ポテンシャル V(r) = -e^2/(4\pi\epsilon_0 r) の下で解くと、波動関数

\psi_{nlm}(r, \theta, \phi) = R_{nl}(r) Y_{lm}(\theta, \phi) \tag{24.7}

が得られる。ここで n, l, m は主量子数・軌道角運動量量子数・磁気量子数。エネルギーは（非相対論的・スピン無視）

E_n = -\frac{13.6\,\text{eV}}{n^2} \tag{24.8}

Bohr 模型と 同じエネルギー だが、構造が深い：

  1. 縮退：n が同じで l, m が違う状態はエネルギーが等しい（Coulomb ポテンシャル固有の縮退）
  2. 波動関数：電子の存在確率分布が空間的に広がっている
  3. 角運動量量子化：L^2 = l(l+1)\hbar^2（Bohr の L = n\hbar は不正確だった）

問い：Bohr 模型と Schrödinger 方程式は同じエネルギーを与えるのに、なぜ Schrödinger 方程式が「正しい」量子力学なのか？

短答：Bohr 模型は エネルギーは正しいが波動関数を持たない。次節で計算する 振動子強度 f_{lu}（観測される線の強度を決める）は波動関数の積分で決まる。観測される線強度を再現するには、波動関数つき Schrödinger 方程式が必要。

もう一歩：相対論的補正（微細構造、Lamb シフト）も Schrödinger 方程式を Dirac 方程式 → QED へ拡張することで導出される。これらの精密構造も観測される（次節以降）

21.4 振動子強度 f_{lu} ― 観測線強度の起源

[本文目安：B3]

中心地図 \alpha_\nu = (\pi e^2/m_e c) f_{lu} n_l \phi(\nu-\nu_0) の 振動子強度 f_{lu} は、量子力学では 遷移行列要素 から計算される：

f_{lu} = \frac{2 m_e \omega_{ul}}{3 \hbar} |\langle u | \mathbf{r} | l \rangle|^2 \cdot \frac{1}{g_l} \tag{24.9}

ここで \langle u | \mathbf{r} | l \rangle は 電気双極子遷移行列要素、\omega_{ul} = (E_u - E_l)/\hbar。物理的解釈：

  • f_{lu} は古典的振動子の 強度を表す無次元量 で、\sum_u f_{lu} = N_\text{electrons}（Thomas-Reiche-Kuhn 和則）
  • 電気双極子の振動が「強い」遷移ほど f_{lu} が大きく、観測される線が強い
  • 遷移行列要素は波動関数の重なりで決まる ― Bohr 模型では計算不可能
ノート

用語：\pi e^2 / m_e c という前因子は 古典 Lorentz 振動子の積分吸収断面積（次元は cm^2 \cdot Hz）に対応する量で、振動数積分した吸収係数 \int \sigma\, d\nu = \pi e^2 f_{lu}/(m_e c) の形で現れる。線形 \phi(\nu-\nu_0) を掛けて積分すると規格化条件 \int \phi\, d\nu = 1 により全積分が \pi e^2 f_{lu}/(m_e c) に戻る ― 古典 + 量子論の橋 の典型例（Thomson 散乱断面積 \sigma_T = 8\pi r_e^2/3 とは別物なので混同しないこと）

対応（観測）：f_{lu} の値は理論計算・実験室で精密に求められており、原子物理の標準データ（NIST Atomic Spectra Database 等）として整備されている。観測線強度 → 柱密度の換算（第20章 curve of growth）にはこのデータが必須

21.4b \pi e^2/m_e c の逆引き ― 古典 Lorentz 振動子から

[本文目安：B3／導出：M1]

中心地図 \alpha_\nu = (\pi e^2/m_e c) f_{lu} n_l \phi(\nu-\nu_0) の 前因子 \pi e^2/m_e c は、第VII部の中心地図のうち、まだ逆引きを明示していなかった因子である。本節でその起源を、古典 Lorentz 振動子（damped harmonic oscillator）モデルから示し、Einstein 係数との関係を通じて量子論との橋を完成させる。

古典 Lorentz 振動子の吸収断面積

電場 \mathbf{E}(t) = \mathbf{E}_0 e^{-i\omega t} に弱く減衰する束縛電子（質量 m_e、電荷 -e、固有角振動数 \omega_0、減衰率 \gamma）の運動方程式：

\ddot{\mathbf{r}} + \gamma \dot{\mathbf{r}} + \omega_0^2 \mathbf{r} = -\frac{e}{m_e}\mathbf{E}(t) \tag{24.10}

定常解の振幅から、放電子による双極子モーメント \mathbf{p}(t) を求め、これに対する 吸収断面積（時間平均パワー吸収率を入射エネルギーフラックスで割ったもの）を計算すると、振動数積分された値は

\int_0^\infty \sigma_\text{cl}(\nu)\, d\nu = \frac{\pi e^2}{m_e c} \tag{24.11}

ここで前因子 \pi e^2/m_e c は 減衰率 \gamma にも線形ぴーくの位置にもよらない普遍量。古典電磁気の力学パラメータのみで決まる。

量子論への橋 ― 振動子強度 f_{lu} の登場

実際の原子は古典振動子ではなく、量子的な多準位系である。各遷移 l \to u の 実効的な吸収強度 は、古典的振動子の振動数積分断面積に 無次元の補正係数 f_{lu}（振動子強度、第21.4節）を掛けたものとして書ける：

\int_0^\infty \sigma_{lu}(\nu)\, d\nu = \frac{\pi e^2}{m_e c} \cdot f_{lu} \tag{24.12}

f_{lu} は量子力学の遷移行列要素 式 24.9 から計算される。Thomas-Reiche-Kuhn 和則 \sum_u f_{lu} = N_\text{electrons} は、「すべての遷移を足し合わせると、古典的に予想される総吸収量に戻る」という意味で、古典-量子対応を定量的に保証する。

線形 \phi(\nu-\nu_0)（規格化 \int \phi\, d\nu = 1）を使って、吸収断面積を振動数依存の形に展開：

\sigma_{lu}(\nu) = \frac{\pi e^2}{m_e c} f_{lu}\, \phi(\nu - \nu_0) \tag{24.13}

これを 柱密度 × 断面積 = 吸収係数 の関係に代入して、中心地図

\boxed{ \alpha_\nu = \frac{\pi e^2}{m_e c}\, f_{lu}\, n_l\, \phi(\nu - \nu_0) } \tag{24.14}

を再現する（第VII部冒頭で経験式として受け入れていたこの式が、いま完全に逆引きされた）。

Einstein B 係数との換算

第20章 §20.2 で導入した Einstein 関係 \alpha_\nu = (h\nu_0/4\pi) n_l B_{lu} \phi(\nu-\nu_0) と本節の 式 24.14 を比べると、両者は 同一の物理量を別の言葉で書いたもの であり、

B_{lu} = \frac{4\pi^2 e^2}{m_e c h\nu_0}\, f_{lu} \tag{24.15}

の換算で結ばれる（J_\nu 規約、§11.3）。さらに Einstein 関係 A_{ul} = (2h\nu^3/c^2) B_{ul} = (2h\nu^3/c^2)(g_l/g_u) B_{lu} と組合せて、自発放出率 A_{ul} も振動子強度から：

A_{ul} = \frac{8\pi^2 e^2 \nu_0^2}{m_e c^3}\, \frac{g_l}{g_u}\, f_{lu} \tag{24.16}

ノート

第VII部中心地図の完成：これで線スペクトル中心地図の全因子（\pi e^2/m_e c, f_{lu}, n_l, \phi, \nu_0）が逆引きされた：

因子 起源 章
\pi e^2 / m_e c 古典 Lorentz 振動子の積分吸収断面積 本節（§21.4b）
f_{lu} 遷移行列要素 |\langle u|\mathbf{r}|l\rangle|^2（量子論的補正） §21.4
n_l LTE（Boltzmann/Saha）または non-LTE 統計平衡 §20.4-§20.5
\phi(\nu - \nu_0) 自然幅・Doppler・衝突幅・Voigt 第22章
\nu_0 原子・分子のエネルギー準位 §21.1-§21.3

連続スペクトル側（第14章で逆引き完了）と並んで、第VII部の中心地図も完全に逆引きされた。第VIII部の QED で両中心地図が統一されるとき、\pi e^2/m_e c と 2h\nu^3/c^2 の両前因子が、同じ電磁場–物質相互作用 Hamiltonian から生じることが明らかになる

21.5 選択則と禁制線 ― なぜ「許されない」遷移が見えるか

[本文目安：B3]

電気双極子遷移は、波動関数のパリティと角運動量の選択則を持つ：

\Delta l = \pm 1, \quad \Delta m = 0, \pm 1 \tag{24.17}

これらを満たさない遷移は、電気双極子では起こらない ― 「禁制」（forbidden）。

しかし、より高次の遷移は弱いながら可能：

  • 電気四重極遷移（E2）：f \sim \alpha^2 f_{E1} \sim 10^{-4} 倍
  • 磁気双極子遷移（M1）：f \sim \alpha^2 f_{E1} \sim 10^{-4} 倍

これらの弱い遷移は、地上の実験室では 衝突脱励起 で先に消されて観測できないが、極低密度の宇宙環境 では時間が許されて自発放出が起こる。

第19章 §19.3 で触れた 禁制線（[O III] $$5007 等）の正体はこれ。

問い：[O III] $5007 は「禁制」と呼ばれるが、HII 領域や惑星状星雲では H$ よりも強く観測されることがある。なぜか？

短答：HII 領域の電子密度（n_e \sim 10^2〜10^4 cm^{-3}）では衝突脱励起時間が長く、A_{ul} \sim 10^{-2} s^{-1} の禁制線でも自発放出が間に合う。一方、[O III] のエネルギー準位構造は 衝突励起断面積が大きく、励起される割合も高い。両者の組合せで、許容線と同程度の強度が出る。

もう一歩：[O III] $$5007 の強度は、星雲の 電子温度 に強く依存し、§23.3 の温度診断に使われる。「禁制」という言葉は、地上実験室での測りにくさを反映するが、宇宙物理では主役級の道具

21.6 多電子原子と分子 ― 水素原子の枠組みの拡張

[本文目安：B3-B4]

水素以外の原子では、複数の電子が同時に存在するため、\psi は単一粒子波動関数の積では書けない（Pauli の排他律と電子間相互作用）。代わりに：

  • Hartree-Fock 近似：各電子の波動関数を、他電子の平均場の中で解く
  • 項記号（term symbol）：原子の状態を ^{2S+1}L_J で表す（例：太陽 D 線の上準位は ^2P_{3/2} と ^2P_{1/2}）
  • 微細構造：スピン-軌道結合による準位分裂、線分裂の起源
  • 超微細構造：核スピンとの結合、21 cm 線の起源（§19.4）

分子では、回転・振動準位 が加わる：

  • 回転準位：E_J = \hbar^2 J(J+1) / 2I、I は慣性モーメント。電波・サブミリ波
  • 振動準位：E_v = \hbar \omega (v + 1/2)、\omega は分子振動振動数。中間赤外
  • 電子準位：原子と同類。紫外〜可視

これらの組合せで、分子線スペクトルの 巨大なライン群（rotational band 等） が生まれる。

注意（観測）：分子線スペクトルは原子線より遥かに複雑だが、回転定数 B と振動定数 \omega_e の 二つの定数 で大半が記述できる。NIST Molecular Spectroscopy Database 等で公開されており、観測者は分子線同定とパラメータ決定にこれを利用する

問題 21-1　Rydberg 公式と Hα の波長

[★ 難易度：☆☆ ] [導出補完]

§21.1・21.2 の Rydberg 公式 \dfrac{1}{\lambda}=R_H\left(\dfrac{1}{n_l^2}-\dfrac{1}{n_u^2}\right)、R_H=1.097\times10^7 m^{-1}、Bohr エネルギー E_n=-13.6/n^2 eV。

  1. h\nu=E_u-E_l から Rydberg 公式の形が出ることを示せ。
  2. Balmer 系列の H\alpha（n_l=2\to n_u=3）の波長を計算し、656 nm を確かめよ。
  3. Lyman 系列の系列極限（n_u\to\infty、n_l=1）の波長を求め、これが水素の電離エネルギー 13.6 eV に対応することを確かめよ。

関連：§21.1 Rydberg 公式、§21.2 Bohr 模型／準位の量子力学的起源は§12.1、Boltzmann 占有は演習 7-4。

模範解答（問題 21-1）

(1) h\nu=E_u-E_l=-\dfrac{13.6}{n_u^2}-\left(-\dfrac{13.6}{n_l^2}\right)=13.6\,\text{eV}\left(\dfrac{1}{n_l^2}-\dfrac{1}{n_u^2}\right)。\nu=c/\lambda より \dfrac{1}{\lambda}=\dfrac{13.6\,\text{eV}}{hc}\left(\dfrac{1}{n_l^2}-\dfrac{1}{n_u^2}\right)=R_H\left(\dfrac{1}{n_l^2}-\dfrac{1}{n_u^2}\right)。

(2) \dfrac{1}{\lambda}=1.097\times10^7\left(\dfrac14-\dfrac19\right)=1.097\times10^7\times0.1389=1.524\times10^6 m^{-1}。\lambda=6.56\times10^{-7} m =656 nm。✓（太陽 Fraunhofer C 線）

(3) n_u\to\infty、n_l=1：\dfrac{1}{\lambda}=R_H\times1=1.097\times10^7 m^{-1}、\lambda=91.2 nm。対応エネルギー E=hc/\lambda=1.986\times10^{-25}/9.12\times10^{-8}=2.18\times10^{-18} J =13.6 eV。これは基底状態（n=1）からの電離エネルギーに一致。

答え：h\nu=E_u-E_l から Rydberg 公式。H\alpha＝656 nm。Lyman 極限 91.2 nm＝13.6 eV（電離エネルギー）。

問題 21-2　Einstein 係数と振動子強度の換算

[★ 難易度：☆☆ ] [導出補完]

§21.4b は中心地図の前因子 \pi e^2/m_ec と Einstein 係数を結ぶ換算を与えた。導出を埋める。中心地図 \alpha_\nu=\dfrac{\pi e^2}{m_ec}f_{lu}n_l\phi と Einstein 形 \alpha_\nu=\dfrac{h\nu_0}{4\pi}n_lB_{lu}\phi（誘導放出補正を当面無視）。

  1. 両式を等置して B_{lu}=\dfrac{4\pi^2e^2}{m_ec\,h\nu_0}f_{lu}（J_\nu 規約）を導け。
  2. Einstein 関係 A_{ul}=\dfrac{2h\nu_0^3}{c^2}B_{ul}=\dfrac{2h\nu_0^3}{c^2}\dfrac{g_l}{g_u}B_{lu} と組合せ、A_{ul}=\dfrac{8\pi^2e^2\nu_0^2}{m_ec^3}\dfrac{g_l}{g_u}f_{lu} を導け。
  3. \pi e^2/m_ec が「古典 Lorentz 振動子の積分吸収断面積」であり、f_{lu} がその量子論的補正（Thomas-Reiche-Kuhn 和則 \sum_u f_{lu}=N_{\rm electrons}）であることの意味を一言で述べよ。

関連：§21.4b 前因子の逆引き、§21.4 振動子強度／Einstein 関係は演習 11-1、QED からの A_{ul} は§25.2。

模範解答（問題 21-2）

(1) 両式の n_l\phi を消して係数を等置：\dfrac{\pi e^2}{m_ec}f_{lu}=\dfrac{h\nu_0}{4\pi}B_{lu}。B_{lu} について解く：

B_{lu}=\frac{4\pi}{h\nu_0}\cdot\frac{\pi e^2}{m_ec}f_{lu}=\frac{4\pi^2e^2}{m_ec\,h\nu_0}f_{lu}.

(2) B_{ul}=（g_l/g_u)B_{lu} を A_{ul}=\dfrac{2h\nu_0^3}{c^2}B_{ul} に代入：

A_{ul}=\frac{2h\nu_0^3}{c^2}\frac{g_l}{g_u}\frac{4\pi^2e^2}{m_ec\,h\nu_0}f_{lu}=\frac{8\pi^2e^2\nu_0^2}{m_ec^3}\frac{g_l}{g_u}f_{lu}.

（h が消え、\nu_0^3/\nu_0=\nu_0^2。）

(3) \pi e^2/m_ec は古典の減衰調和振動子（Lorentz 振動子）が与える振動数積分吸収断面積で、減衰率にもピーク位置にもよらない普遍量。実際の原子はこれに無次元の量子補正 f_{lu} を掛けたもので、全遷移の和 \sum_u f_{lu}=N_{\rm electrons}（TRK 和則）が「すべて足すと古典総吸収量に戻る」ことを保証する。古典-量子の橋。

答え：B_{lu}=\dfrac{4\pi^2e^2}{m_ech\nu_0}f_{lu}、A_{ul}=\dfrac{8\pi^2e^2\nu_0^2}{m_ec^3}\dfrac{g_l}{g_u}f_{lu}。\pi e^2/m_ec＝古典積分断面積、f_{lu}＝量子補正（TRK 和則）。

問題 21-3　選択則と禁制線の次数

[★ 難易度：☆☆ ] [理解を深める]

§21.5 の電気双極子選択則 \Delta l=\pm1、\Delta m=0,\pm1 と禁制遷移を読む。

  1. 電気双極子（E1）選択則を満たさない遷移が「禁制」と呼ばれる理由を、遷移行列要素 \langle u|\mathbf r|l\rangle の観点から述べよ。
  2. 磁気双極子（M1）・電気四重極（E2）遷移の強度が E1 の約 \alpha^2\sim10^{-4} 倍であること（\alpha\simeq1/137）を確かめ、\alpha^2 がどこから来るか一言で述べよ。
  3. 禁制線が地上では見えにくいのに星間で見える理由を、演習 19-2 と結びつけて述べよ。21 cm 線が M1 禁制遷移であることにも触れよ。

関連：§21.5 選択則と禁制線／QED からの選択則は§25.5、禁制線と低密度は演習 19-2、密度診断は§23.3。

模範解答（問題 21-3）

(1) E1 遷移強度は電気双極子行列要素 \langle u|\mathbf r|l\rangle で決まる。パリティと角運動量の保存から、\Delta l=\pm1 等を満たさない遷移ではこの行列要素がゼロになり、最低次（E1）では遷移が起こらない。これが「禁制」。

(2) M1・E2 は次数が一つ高い過程で、典型的に振幅が微細構造定数 \alpha 程度小さく、強度（振幅の2乗）が \alpha^2\simeq(1/137)^2\simeq5.3\times10^{-5}\sim10^{-4} 倍。\alpha は電磁相互作用の結合定数で、より高次の多重極展開の各段で \alpha のべきが効く。

(3) 禁制線は A_{ul} が極端に小さく長寿命。地上の高密度では自発放出より衝突脱励起が先に起こり消える（演習 19-2）。星間の極低密度では衝突が稀で自発放出が間に合い観測できる。21 cm 線は水素基底状態の超微細構造の M1 禁制遷移で、A_{ul}\sim10^{-15} s^{-1} と極小だが膨大な中性水素のおかげで観測される。

答え：E1 選択則違反＝行列要素ゼロ＝禁制。M1/E2 は \alpha^2\sim10^{-4} 倍。低密度の星間でのみ観測（21 cm も M1 禁制）。

問題 21-4　なぜ Bohr 模型では線強度が出ないか

[★ 難易度：☆ ] [理解を深める]

§21.2・21.3 は Bohr 模型と Schrödinger 方程式が同じエネルギーを与えるのに、後者が「正しい」量子力学だと述べた。

  1. Bohr 模型と Schrödinger 方程式が一致する点（エネルギー準位）と、異なる点（波動関数・縮退・角運動量）を述べよ。
  2. 振動子強度 f_{lu}\propto|\langle u|\mathbf r|l\rangle|^2 がなぜ Bohr 模型では計算できないか述べよ。
  3. 観測される線の振動数は Bohr 模型で説明できるが、線の強度には Schrödinger 方程式（波動関数）が必要、という構図を、中心地図 \alpha_\nu の \nu_0 と f_{lu} に対応づけて述べよ。

関連：§21.3 Schrödinger 方程式／固有値問題は§12.1、振動子強度の換算は演習 21-2。

模範解答（問題 21-4）

(1) 一致：エネルギー準位 E_n=-13.6/n^2 eV（線振動数を正しく与える）。異なる：Bohr は電子を許される軌道上に「正確に」置く描像で波動関数を持たず、角運動量を L=n\hbar とする（Schrödinger は L^2=l(l+1)\hbar^2）。Schrödinger は同じ n で l,m の異なる縮退状態と、空間に広がる存在確率分布（波動関数）を持つ。

(2) f_{lu} は上下準位の波動関数の重なり \langle u|\mathbf r|l\rangle=\int\psi_u^*\,\mathbf r\,\psi_l\,d^3r から計算される。Bohr 模型は波動関数を持たないので、この積分を実行できず f_{lu} を予言できない。

(3) 中心地図 \alpha_\nu=(\pi e^2/m_ec)f_{lu}n_l\phi(\nu-\nu_0) で、\nu_0（線振動数＝準位差）は Bohr 模型でも出るが、f_{lu}（線強度を決める振動子強度）は波動関数の行列要素ゆえ Schrödinger 方程式が要る。線の「位置」と「強さ」で必要な理論の深さが違う。

答え：エネルギー（\nu_0）は一致するが、Bohr は波動関数を持たず f_{lu}=|\langle u|\mathbf r|l\rangle|^2 を計算できない。線強度には Schrödinger が必要。

## 21.1 Rydberg 公式 ― 水素線の振動数の起源 {#sec-atomic-rydberg}

[本文目安：B1-B2]

第19章 §19.1 で見た太陽の Fraunhofer C 線（$\lambda = 656$ nm, H$\alpha$）と、それに続く水素線系列の振動数は、すべて **Rydberg 公式** で書ける：

$$
\frac{1}{\lambda} = R_H \left(\frac{1}{n_l^2} - \frac{1}{n_u^2}\right)
$$ {#eq-rydberg-formula}

ここで $R_H \simeq 1.097 \times 10^7$ m$^{-1}$ は Rydberg 定数、$n_l, n_u$ は主量子数。

- **Lyman 系列**（$n_l = 1$）：紫外、$\lambda < 122$ nm
- **Balmer 系列**（$n_l = 2$）：可視、H$\alpha$ = 656 nm, H$\beta$ = 486 nm, ...
- **Paschen 系列**（$n_l = 3$）：近赤外
- **Brackett、Pfund 系列**（$n_l = 4, 5$）：赤外〜遠赤外

これは経験式として 1888 年に発見された。**なぜこの形になるか** を逆引きするのが本節以降の主題。

::: {.callout-note}
**対応（観測）**：観測可能なすべての水素線振動数が **二つの整数 $n_l, n_u$ で記述できる** という事実は、本書の主題「観測スペクトルから量子物理を逆引きする」の典型例。1888 年時点でこの規則性が認識されたが、それが何を意味するかは、Bohr 模型（1913）まで待たねばならなかった
:::

## 21.2 Bohr 模型 ― 線振動数の最初の説明 {#sec-atomic-bohr}

[本文目安：B1-B2]

Bohr（1913）は、Rutherford の原子模型に **角運動量量子化** を導入した：

$$
L = n\hbar, \quad n = 1, 2, 3, \ldots
$$ {#eq-bohr-quantization}

これと古典電磁気の Coulomb 力 + Newton 運動方程式を組み合わせると、エネルギー準位

$$
E_n = -\frac{1}{2} \frac{m_e e^4}{(4\pi\epsilon_0)^2 \hbar^2} \frac{1}{n^2} = -\frac{13.6\,\text{eV}}{n^2}
$$ {#eq-bohr-energy}

が出る。線振動数は $h\nu = E_u - E_l$ から

$$
\frac{1}{\lambda} = \frac{m_e e^4}{(4\pi\epsilon_0)^2 \cdot 4\pi \hbar^3 c} \left(\frac{1}{n_l^2} - \frac{1}{n_u^2}\right) = R_H \left(\frac{1}{n_l^2} - \frac{1}{n_u^2}\right)
$$ {#eq-bohr-rydberg}

これで Rydberg 公式が量子物理から **逆引き** された。

::: {.callout-warning appearance="simple"}
**注意（観測）**：Bohr 模型は **線振動数は完璧に予言する** が、**振動子強度 $f_{lu}$ は予言できない**。なぜなら、模型は電子の位置を許される軌道の上に「正確に」置く描像で、波動関数を持たない。$f_{lu}$ は次節の Schrödinger 方程式と波動関数によって初めて計算できる
:::

## 21.3 Schrödinger 方程式と水素原子の波動関数 {#sec-atomic-schrodinger}

[本文目安：B2-B3]

電子の波動的性質を取り入れた Schrödinger 方程式（1926）を Coulomb ポテンシャル $V(r) = -e^2/(4\pi\epsilon_0 r)$ の下で解くと、波動関数

$$
\psi_{nlm}(r, \theta, \phi) = R_{nl}(r) Y_{lm}(\theta, \phi)
$$ {#eq-hydrogen-wavefunction}

が得られる。ここで $n, l, m$ は主量子数・軌道角運動量量子数・磁気量子数。エネルギーは（非相対論的・スピン無視）

$$
E_n = -\frac{13.6\,\text{eV}}{n^2}
$$ {#eq-schrodinger-energy}

Bohr 模型と **同じエネルギー** だが、構造が深い：

1. **縮退**：$n$ が同じで $l, m$ が違う状態はエネルギーが等しい（Coulomb ポテンシャル固有の縮退）
2. **波動関数**：電子の存在確率分布が空間的に広がっている
3. **角運動量量子化**：$L^2 = l(l+1)\hbar^2$（Bohr の $L = n\hbar$ は不正確だった）

::: {.callout-tip appearance="simple"}
**問い**：Bohr 模型と Schrödinger 方程式は同じエネルギーを与えるのに、なぜ Schrödinger 方程式が「正しい」量子力学なのか？

**短答**：Bohr 模型は **エネルギーは正しいが波動関数を持たない**。次節で計算する **振動子強度 $f_{lu}$**（観測される線の強度を決める）は波動関数の積分で決まる。観測される線強度を再現するには、波動関数つき Schrödinger 方程式が必要。

**もう一歩**：相対論的補正（微細構造、Lamb シフト）も Schrödinger 方程式を Dirac 方程式 → QED へ拡張することで導出される。これらの精密構造も観測される（次節以降）
:::

## 21.4 振動子強度 $f_{lu}$ ― 観測線強度の起源 {#sec-atomic-oscillator-strength}

[本文目安：B3]

中心地図 $\alpha_\nu = (\pi e^2/m_e c) f_{lu} n_l \phi(\nu-\nu_0)$ の **振動子強度** $f_{lu}$ は、量子力学では **遷移行列要素** から計算される：

$$
f_{lu} = \frac{2 m_e \omega_{ul}}{3 \hbar} |\langle u | \mathbf{r} | l \rangle|^2 \cdot \frac{1}{g_l}
$$ {#eq-oscillator-strength}

ここで $\langle u | \mathbf{r} | l \rangle$ は **電気双極子遷移行列要素**、$\omega_{ul} = (E_u - E_l)/\hbar$。物理的解釈：

- $f_{lu}$ は古典的振動子の **強度を表す無次元量** で、$\sum_u f_{lu} = N_\text{electrons}$（**Thomas-Reiche-Kuhn 和則**）
- 電気双極子の振動が「強い」遷移ほど $f_{lu}$ が大きく、観測される線が強い
- 遷移行列要素は波動関数の重なりで決まる ― Bohr 模型では計算不可能

::: {.callout-note}
**用語**：$\pi e^2 / m_e c$ という前因子は **古典 Lorentz 振動子の積分吸収断面積**（次元は cm$^2 \cdot$ Hz）に対応する量で、振動数積分した吸収係数 $\int \sigma\, d\nu = \pi e^2 f_{lu}/(m_e c)$ の形で現れる。線形 $\phi(\nu-\nu_0)$ を掛けて積分すると規格化条件 $\int \phi\, d\nu = 1$ により全積分が $\pi e^2 f_{lu}/(m_e c)$ に戻る ― **古典 + 量子論の橋** の典型例（Thomson 散乱断面積 $\sigma_T = 8\pi r_e^2/3$ とは別物なので混同しないこと）

**対応（観測）**：$f_{lu}$ の値は理論計算・実験室で精密に求められており、原子物理の標準データ（NIST Atomic Spectra Database 等）として整備されている。観測線強度 → 柱密度の換算（第20章 curve of growth）にはこのデータが必須
:::

## 21.4b $\pi e^2/m_e c$ の逆引き ― 古典 Lorentz 振動子から {#sec-atomic-lorentz-prefactor}

[本文目安：B3／導出：M1]

中心地図 $\alpha_\nu = (\pi e^2/m_e c) f_{lu} n_l \phi(\nu-\nu_0)$ の **前因子 $\pi e^2/m_e c$** は、第VII部の中心地図のうち、まだ逆引きを明示していなかった因子である。本節でその起源を、古典 Lorentz 振動子（damped harmonic oscillator）モデルから示し、Einstein 係数との関係を通じて量子論との橋を完成させる。

### 古典 Lorentz 振動子の吸収断面積

電場 $\mathbf{E}(t) = \mathbf{E}_0 e^{-i\omega t}$ に弱く減衰する束縛電子（質量 $m_e$、電荷 $-e$、固有角振動数 $\omega_0$、減衰率 $\gamma$）の運動方程式：

$$
\ddot{\mathbf{r}} + \gamma \dot{\mathbf{r}} + \omega_0^2 \mathbf{r} = -\frac{e}{m_e}\mathbf{E}(t)
$$ {#eq-lorentz-oscillator}

定常解の振幅から、放電子による双極子モーメント $\mathbf{p}(t)$ を求め、これに対する **吸収断面積**（時間平均パワー吸収率を入射エネルギーフラックスで割ったもの）を計算すると、振動数積分された値は

$$
\int_0^\infty \sigma_\text{cl}(\nu)\, d\nu = \frac{\pi e^2}{m_e c}
$$ {#eq-classical-integrated-cross-section}

ここで前因子 $\pi e^2/m_e c$ は **減衰率 $\gamma$ にも線形ぴーくの位置にもよらない普遍量**。古典電磁気の力学パラメータのみで決まる。

### 量子論への橋 ― 振動子強度 $f_{lu}$ の登場

実際の原子は古典振動子ではなく、量子的な多準位系である。各遷移 $l \to u$ の **実効的な吸収強度** は、古典的振動子の振動数積分断面積に **無次元の補正係数** $f_{lu}$（振動子強度、第21.4節）を掛けたものとして書ける：

$$
\int_0^\infty \sigma_{lu}(\nu)\, d\nu = \frac{\pi e^2}{m_e c} \cdot f_{lu}
$$ {#eq-quantum-integrated-cross-section}

$f_{lu}$ は量子力学の遷移行列要素 @eq-oscillator-strength から計算される。Thomas-Reiche-Kuhn 和則 $\sum_u f_{lu} = N_\text{electrons}$ は、「すべての遷移を足し合わせると、古典的に予想される総吸収量に戻る」という意味で、古典-量子対応を定量的に保証する。

線形 $\phi(\nu-\nu_0)$（規格化 $\int \phi\, d\nu = 1$）を使って、吸収断面積を振動数依存の形に展開：

$$
\sigma_{lu}(\nu) = \frac{\pi e^2}{m_e c} f_{lu}\, \phi(\nu - \nu_0)
$$ {#eq-cross-section-line}

これを **柱密度 × 断面積 = 吸収係数** の関係に代入して、中心地図

$$
\boxed{
\alpha_\nu = \frac{\pi e^2}{m_e c}\, f_{lu}\, n_l\, \phi(\nu - \nu_0)
}
$$ {#eq-line-absorption-coefficient-complete}

を再現する（第VII部冒頭で経験式として受け入れていたこの式が、いま完全に逆引きされた）。

### Einstein $B$ 係数との換算

第20章 §20.2 で導入した Einstein 関係 $\alpha_\nu = (h\nu_0/4\pi) n_l B_{lu} \phi(\nu-\nu_0)$ と本節の @eq-line-absorption-coefficient-complete を比べると、両者は **同一の物理量を別の言葉で書いたもの** であり、

$$
B_{lu} = \frac{4\pi^2 e^2}{m_e c h\nu_0}\, f_{lu}
$$ {#eq-B-f-conversion}

の換算で結ばれる（$J_\nu$ 規約、§11.3）。さらに Einstein 関係 $A_{ul} = (2h\nu^3/c^2) B_{ul} = (2h\nu^3/c^2)(g_l/g_u) B_{lu}$ と組合せて、自発放出率 $A_{ul}$ も振動子強度から：

$$
A_{ul} = \frac{8\pi^2 e^2 \nu_0^2}{m_e c^3}\, \frac{g_l}{g_u}\, f_{lu}
$$ {#eq-A-f-conversion}

::: {.callout-note}
**第VII部中心地図の完成**：これで線スペクトル中心地図の全因子（$\pi e^2/m_e c$, $f_{lu}$, $n_l$, $\phi$, $\nu_0$）が逆引きされた：

| 因子 | 起源 | 章 |
|---|---|---|
| $\pi e^2 / m_e c$ | 古典 Lorentz 振動子の積分吸収断面積 | **本節（§21.4b）** |
| $f_{lu}$ | 遷移行列要素 $|\langle u|\mathbf{r}|l\rangle|^2$（量子論的補正） | §21.4 |
| $n_l$ | LTE（Boltzmann/Saha）または non-LTE 統計平衡 | §20.4-§20.5 |
| $\phi(\nu - \nu_0)$ | 自然幅・Doppler・衝突幅・Voigt | 第22章 |
| $\nu_0$ | 原子・分子のエネルギー準位 | §21.1-§21.3 |

連続スペクトル側（第14章で逆引き完了）と並んで、**第VII部の中心地図も完全に逆引きされた**。第VIII部の QED で両中心地図が統一されるとき、$\pi e^2/m_e c$ と $2h\nu^3/c^2$ の両前因子が、同じ電磁場–物質相互作用 Hamiltonian から生じることが明らかになる
:::

## 21.5 選択則と禁制線 ― なぜ「許されない」遷移が見えるか {#sec-atomic-selection-rules}

[本文目安：B3]

電気双極子遷移は、波動関数のパリティと角運動量の選択則を持つ：

$$
\Delta l = \pm 1, \quad \Delta m = 0, \pm 1
$$ {#eq-selection-rules-E1}

これらを満たさない遷移は、電気双極子では起こらない ― 「**禁制**」（**forbidden**）。

しかし、より高次の遷移は弱いながら可能：

- **電気四重極遷移**（**E2**）：$f \sim \alpha^2 f_{E1} \sim 10^{-4}$ 倍
- **磁気双極子遷移**（**M1**）：$f \sim \alpha^2 f_{E1} \sim 10^{-4}$ 倍

これらの弱い遷移は、地上の実験室では **衝突脱励起** で先に消されて観測できないが、**極低密度の宇宙環境** では時間が許されて自発放出が起こる。

第19章 §19.3 で触れた **禁制線**（[O III] $\lambda$5007 等）の正体はこれ。

::: {.callout-tip appearance="simple"}
**問い**：[O III] $\lambda$5007 は「禁制」と呼ばれるが、HII 領域や惑星状星雲では H$\beta$ よりも強く観測されることがある。なぜか？

**短答**：HII 領域の電子密度（$n_e \sim 10^2$〜$10^4$ cm$^{-3}$）では衝突脱励起時間が長く、$A_{ul} \sim 10^{-2}$ s$^{-1}$ の禁制線でも自発放出が間に合う。一方、[O III] のエネルギー準位構造は **衝突励起断面積が大きく**、励起される割合も高い。両者の組合せで、許容線と同程度の強度が出る。

**もう一歩**：[O III] $\lambda$5007 の強度は、星雲の **電子温度** に強く依存し、§23.3 の温度診断に使われる。「禁制」という言葉は、地上実験室での測りにくさを反映するが、宇宙物理では主役級の道具
:::

## 21.6 多電子原子と分子 ― 水素原子の枠組みの拡張 {#sec-atomic-multi-electron}

[本文目安：B3-B4]

水素以外の原子では、複数の電子が同時に存在するため、$\psi$ は単一粒子波動関数の積では書けない（Pauli の排他律と電子間相互作用）。代わりに：

- **Hartree-Fock 近似**：各電子の波動関数を、他電子の平均場の中で解く
- **項記号**（**term symbol**）：原子の状態を $^{2S+1}L_J$ で表す（例：太陽 D 線の上準位は $^2P_{3/2}$ と $^2P_{1/2}$）
- **微細構造**：スピン-軌道結合による準位分裂、線分裂の起源
- **超微細構造**：核スピンとの結合、21 cm 線の起源（§19.4）

分子では、**回転・振動準位** が加わる：

- **回転準位**：$E_J = \hbar^2 J(J+1) / 2I$、$I$ は慣性モーメント。電波・サブミリ波
- **振動準位**：$E_v = \hbar \omega (v + 1/2)$、$\omega$ は分子振動振動数。中間赤外
- **電子準位**：原子と同類。紫外〜可視

これらの組合せで、分子線スペクトルの **巨大なライン群（rotational band 等）** が生まれる。

::: {.callout-warning appearance="simple"}
**注意（観測）**：分子線スペクトルは原子線より遥かに複雑だが、回転定数 $B$ と振動定数 $\omega_e$ の **二つの定数** で大半が記述できる。NIST Molecular Spectroscopy Database 等で公開されており、観測者は分子線同定とパラメータ決定にこれを利用する
:::

### 問題 21-1　Rydberg 公式と Hα の波長 {#ex-21-1 .unnumbered}

[★ 難易度：☆☆ ] [tag:導出補完]

§21.1・21.2 の Rydberg 公式 $\dfrac{1}{\lambda}=R_H\left(\dfrac{1}{n_l^2}-\dfrac{1}{n_u^2}\right)$、$R_H=1.097\times10^7$ m$^{-1}$、Bohr エネルギー $E_n=-13.6/n^2$ eV。

1. $h\nu=E_u-E_l$ から Rydberg 公式の形が出ることを示せ。
2. Balmer 系列の H$\alpha$（$n_l=2\to n_u=3$）の波長を計算し、656 nm を確かめよ。
3. Lyman 系列の系列極限（$n_u\to\infty$、$n_l=1$）の波長を求め、これが水素の電離エネルギー 13.6 eV に対応することを確かめよ。

**関連**：[§21.1 Rydberg 公式](#sec-atomic-rydberg)、[§21.2 Bohr 模型](#sec-atomic-bohr)／準位の量子力学的起源は[§12.1](../part4/12-quantum-foundations.qmd#sec-quantum-foundations-states)、Boltzmann 占有は[演習 7-4](../part3/07-statistical-mechanics.qmd#ex-7-4)。

::: {.callout-derive collapse="true"}
## 模範解答（問題 21-1）

**(1)** $h\nu=E_u-E_l=-\dfrac{13.6}{n_u^2}-\left(-\dfrac{13.6}{n_l^2}\right)=13.6\,\text{eV}\left(\dfrac{1}{n_l^2}-\dfrac{1}{n_u^2}\right)$。$\nu=c/\lambda$ より $\dfrac{1}{\lambda}=\dfrac{13.6\,\text{eV}}{hc}\left(\dfrac{1}{n_l^2}-\dfrac{1}{n_u^2}\right)=R_H\left(\dfrac{1}{n_l^2}-\dfrac{1}{n_u^2}\right)$。

**(2)** $\dfrac{1}{\lambda}=1.097\times10^7\left(\dfrac14-\dfrac19\right)=1.097\times10^7\times0.1389=1.524\times10^6$ m$^{-1}$。$\lambda=6.56\times10^{-7}$ m $=656$ nm。✓（太陽 Fraunhofer C 線）

**(3)** $n_u\to\infty$、$n_l=1$：$\dfrac{1}{\lambda}=R_H\times1=1.097\times10^7$ m$^{-1}$、$\lambda=91.2$ nm。対応エネルギー $E=hc/\lambda=1.986\times10^{-25}/9.12\times10^{-8}=2.18\times10^{-18}$ J $=13.6$ eV。これは基底状態（$n=1$）からの電離エネルギーに一致。

**答え**：$h\nu=E_u-E_l$ から Rydberg 公式。H$\alpha$＝656 nm。Lyman 極限 91.2 nm＝13.6 eV（電離エネルギー）。
:::

### 問題 21-2　Einstein 係数と振動子強度の換算 {#ex-21-2 .unnumbered}

[★ 難易度：☆☆ ] [tag:導出補完]

§21.4b は中心地図の前因子 $\pi e^2/m_ec$ と Einstein 係数を結ぶ換算を与えた。導出を埋める。中心地図 $\alpha_\nu=\dfrac{\pi e^2}{m_ec}f_{lu}n_l\phi$ と Einstein 形 $\alpha_\nu=\dfrac{h\nu_0}{4\pi}n_lB_{lu}\phi$（誘導放出補正を当面無視）。

1. 両式を等置して $B_{lu}=\dfrac{4\pi^2e^2}{m_ec\,h\nu_0}f_{lu}$（$J_\nu$ 規約）を導け。
2. Einstein 関係 $A_{ul}=\dfrac{2h\nu_0^3}{c^2}B_{ul}=\dfrac{2h\nu_0^3}{c^2}\dfrac{g_l}{g_u}B_{lu}$ と組合せ、$A_{ul}=\dfrac{8\pi^2e^2\nu_0^2}{m_ec^3}\dfrac{g_l}{g_u}f_{lu}$ を導け。
3. $\pi e^2/m_ec$ が「古典 Lorentz 振動子の積分吸収断面積」であり、$f_{lu}$ がその量子論的補正（Thomas-Reiche-Kuhn 和則 $\sum_u f_{lu}=N_{\rm electrons}$）であることの意味を一言で述べよ。

## 模範解答（問題 21-2）

**(1)** 両式の $n_l\phi$ を消して係数を等置：$\dfrac{\pi e^2}{m_ec}f_{lu}=\dfrac{h\nu_0}{4\pi}B_{lu}$。$B_{lu}$ について解く：

$$B_{lu}=\frac{4\pi}{h\nu_0}\cdot\frac{\pi e^2}{m_ec}f_{lu}=\frac{4\pi^2e^2}{m_ec\,h\nu_0}f_{lu}.$$

**(2)** $B_{ul}=（g_l/g_u)B_{lu}$ を $A_{ul}=\dfrac{2h\nu_0^3}{c^2}B_{ul}$ に代入：

$$A_{ul}=\frac{2h\nu_0^3}{c^2}\frac{g_l}{g_u}\frac{4\pi^2e^2}{m_ec\,h\nu_0}f_{lu}=\frac{8\pi^2e^2\nu_0^2}{m_ec^3}\frac{g_l}{g_u}f_{lu}.$$

（$h$ が消え、$\nu_0^3/\nu_0=\nu_0^2$。）

**(3)** $\pi e^2/m_ec$ は古典の減衰調和振動子（Lorentz 振動子）が与える振動数積分吸収断面積で、減衰率にもピーク位置にもよらない普遍量。実際の原子はこれに無次元の量子補正 $f_{lu}$ を掛けたもので、全遷移の和 $\sum_u f_{lu}=N_{\rm electrons}$（TRK 和則）が「すべて足すと古典総吸収量に戻る」ことを保証する。古典-量子の橋。

**答え**：$B_{lu}=\dfrac{4\pi^2e^2}{m_ech\nu_0}f_{lu}$、$A_{ul}=\dfrac{8\pi^2e^2\nu_0^2}{m_ec^3}\dfrac{g_l}{g_u}f_{lu}$。$\pi e^2/m_ec$＝古典積分断面積、$f_{lu}$＝量子補正（TRK 和則）。

### 問題 21-3　選択則と禁制線の次数 {#ex-21-3 .unnumbered}

[★ 難易度：☆☆ ] [tag:理解を深める]

1. 電気双極子（E1）選択則を満たさない遷移が「禁制」と呼ばれる理由を、遷移行列要素 $\langle u|\mathbf r|l\rangle$ の観点から述べよ。
2. 磁気双極子（M1）・電気四重極（E2）遷移の強度が E1 の約 $\alpha^2\sim10^{-4}$ 倍であること（$\alpha\simeq1/137$）を確かめ、$\alpha^2$ がどこから来るか一言で述べよ。
3. 禁制線が地上では見えにくいのに星間で見える理由を、演習 19-2 と結びつけて述べよ。21 cm 線が M1 禁制遷移であることにも触れよ。

**関連**：[§21.5 選択則と禁制線](#sec-atomic-selection-rules)／QED からの選択則は[§25.5](../part8/25-qed-minimum.qmd#sec-qed-minimum-selection)、禁制線と低密度は[演習 19-2](../part7/19-where-lines.qmd#ex-19-2)、密度診断は[§23.3](../part7/23-line-applications.qmd#sec-line-applications-diagnostics)。

::: {.callout-derive collapse="true"}
## 模範解答（問題 21-3）

**(1)** E1 遷移強度は電気双極子行列要素 $\langle u|\mathbf r|l\rangle$ で決まる。パリティと角運動量の保存から、$\Delta l=\pm1$ 等を満たさない遷移ではこの行列要素がゼロになり、最低次（E1）では遷移が起こらない。これが「禁制」。

**(2)** M1・E2 は次数が一つ高い過程で、典型的に振幅が微細構造定数 $\alpha$ 程度小さく、強度（振幅の2乗）が $\alpha^2\simeq(1/137)^2\simeq5.3\times10^{-5}\sim10^{-4}$ 倍。$\alpha$ は電磁相互作用の結合定数で、より高次の多重極展開の各段で $\alpha$ のべきが効く。

**(3)** 禁制線は $A_{ul}$ が極端に小さく長寿命。地上の高密度では自発放出より衝突脱励起が先に起こり消える（[演習 19-2](../part7/19-where-lines.qmd#ex-19-2)）。星間の極低密度では衝突が稀で自発放出が間に合い観測できる。21 cm 線は水素基底状態の超微細構造の M1 禁制遷移で、$A_{ul}\sim10^{-15}$ s$^{-1}$ と極小だが膨大な中性水素のおかげで観測される。

**答え**：E1 選択則違反＝行列要素ゼロ＝禁制。M1/E2 は $\alpha^2\sim10^{-4}$ 倍。低密度の星間でのみ観測（21 cm も M1 禁制）。
:::

### 問題 21-4　なぜ Bohr 模型では線強度が出ないか {#ex-21-4 .unnumbered}

[★ 難易度：☆ ] [tag:理解を深める]

§21.2・21.3 は Bohr 模型と Schrödinger 方程式が同じエネルギーを与えるのに、後者が「正しい」量子力学だと述べた。

1. Bohr 模型と Schrödinger 方程式が一致する点（エネルギー準位）と、異なる点（波動関数・縮退・角運動量）を述べよ。
2. 振動子強度 $f_{lu}\propto|\langle u|\mathbf r|l\rangle|^2$ がなぜ Bohr 模型では計算できないか述べよ。
3. 観測される線の**振動数**は Bohr 模型で説明できるが、線の**強度**には Schrödinger 方程式（波動関数）が必要、という構図を、中心地図 $\alpha_\nu$ の $\nu_0$ と $f_{lu}$ に対応づけて述べよ。

**関連**：[§21.3 Schrödinger 方程式](#sec-atomic-schrodinger)／固有値問題は[§12.1](../part4/12-quantum-foundations.qmd#sec-quantum-foundations-states)、振動子強度の換算は[演習 21-2](#ex-21-2)。

::: {.callout-derive collapse="true"}
## 模範解答（問題 21-4）

**(1)** 一致：エネルギー準位 $E_n=-13.6/n^2$ eV（線振動数を正しく与える）。異なる：Bohr は電子を許される軌道上に「正確に」置く描像で波動関数を持たず、角運動量を $L=n\hbar$ とする（Schrödinger は $L^2=l(l+1)\hbar^2$）。Schrödinger は同じ $n$ で $l,m$ の異なる縮退状態と、空間に広がる存在確率分布（波動関数）を持つ。

**(2)** $f_{lu}$ は上下準位の波動関数の重なり $\langle u|\mathbf r|l\rangle=\int\psi_u^*\,\mathbf r\,\psi_l\,d^3r$ から計算される。Bohr 模型は波動関数を持たないので、この積分を実行できず $f_{lu}$ を予言できない。

**(3)** 中心地図 $\alpha_\nu=(\pi e^2/m_ec)f_{lu}n_l\phi(\nu-\nu_0)$ で、$\nu_0$（線振動数＝準位差）は Bohr 模型でも出るが、$f_{lu}$（線強度を決める振動子強度）は波動関数の行列要素ゆえ Schrödinger 方程式が要る。線の「位置」と「強さ」で必要な理論の深さが違う。

**答え**：エネルギー（$\nu_0$）は一致するが、Bohr は波動関数を持たず $f_{lu}=|\langle u|\mathbf r|l\rangle|^2$ を計算できない。線強度には Schrödinger が必要。
:::

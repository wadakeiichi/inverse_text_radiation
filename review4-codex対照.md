## (b) Codex 新規指摘の裁定（6件、全採用）

### 1. 誘導放出補正の章間不整合（Codex #1・#2・#3、同一問題群）

**指摘**: 線吸収係数の中心地図 `_includes/_hydrogen-map.qmd:16`、その再現 `part7/22-atomic-physics.qmd:188-198`、`part7/23-line-shape.qmd:110` の α_ν がいずれも誘導放出補正 [1−(g_l n_u)/(g_u n_l)] を含まない一方、`part7/21-line-formation.qmd:84` は補正込みで α_ν を定義しており、**同じ記号 α_ν が場所により正味吸収／真吸収の別物を指す**。さらに 22:198 は「第21章 §21.2 で導入した Einstein 関係」として 21:84 の式を**括弧なしで引用**している（厳密には不正確な引用）。

**裁定過程（python3 検算）**: LTE での補正因子 [1−e^(−hν/kT)] は
- Hα（恒星大気 5800 K）: 0.977（2%効果、可視ではほぼ無視可）
- Lyα（10⁴ K）: 1.000
- 21 cm（100 K）: 6.8×10⁻⁴（**吸収を3桁落とす。本質的**）
- CO(1−0) 115 GHz（20 K）: 0.758→補正24%（分子線で本質的）

地図の「適用範囲」注記は E1/M1/禁制線の前因子問題のみで誘導放出には触れていない。地図は 21 cm 線を明示的に扱うので、電波域で3桁効く補正の欠落は単なる省略では済まない。なお 23:115 の弱線等価幅の係数 πe²λ²/(m_ec²) 自体は Draine 標準形と一致（検算済み）で、可視・UV 吸収線の成長曲線文脈では n_u≈0 が標準近似のため、23:110 は注記のみでよい。

**採否**: **採用**。地図＋22章は[中]（多章に波及する中心地図の整合問題。review3 の _planck-map 因子表と同格）、23:110 は[軽微]に降格（Codex は中と評価したが、係数は正しく文脈上 n_u≈0 が標準のため）。

### 2. 禁制線と LTE の過剰一般化（Codex #4）

**指摘**: `part7/21-line-formation.qmd:192`（Codex 表記では 189）「禁制線（小さい A_ul）は n_crit が低く、**希薄な星雲でも LTE 的に振る舞いやすい**」。

**裁定過程**: 比較としての核（低い n_crit → より低い密度で熱平衡化）は正しく、Codex の「因果が逆」という表現は強すぎる。しかし絶対的な主張としては誤り：典型的な電離星雲は n_e〜10²–10⁴ cm⁻³ に対し、n_crit は [O III]λ5007 で 6.8×10⁵、[N II]λ6583 で 8.7×10⁴ cm⁻³（Osterbrock & Ferland の標準値）であり、主要な星雲禁制線は n_e≪n_crit の **non-LTE（サブサーマル）側**にある。さらに直後の文が引く [S II]λ6716/6731 密度診断（§24.3）は n_e〜n_crit の**遷移域でこそ**線比が密度に感応する ― LTE 極限（n_e≫n_crit）では線比は Boltzmann 値に飽和して密度感度を失う。つまり「LTE 的に振る舞いやすい」は、その直後に紹介する診断の動作原理と矛盾する。

**採否**: **採用[中]**。review4 §3-1（24:131、密度感度の因果逆転）と同じ問題群の**別箇所**。§3-1 と併せて「禁制線・臨界密度まわりの説明」を一括修正するのが効率的。修正案：「禁制線は n_crit が低く、より低い密度で熱平衡化に近づく。ただし典型的な電離星雲（n_e〜10²–10⁴ cm⁻³）では多くの禁制線はなお n_e≲n_crit の non-LTE 側にあり、この遷移域にあることこそが §24.3 の密度診断の前提となる」等。

### 3. 相対論的円盤の観測フラックス式の振動数変換落ち（Codex #6）

**指摘**: `part6/18-accretion-high-energy.qmd:216` の @eq-grdisk-flux「F_ν^obs = ∫ B_ν(T(R,φ))·g³·dA_proper」は、赤方偏移因子 g³ を掛けながら Planck 関数を**観測振動数のまま**評価しており、不変量 I_ν/ν³ から出る正しい形 g³ B_{ν_obs/g}(T)（＝B_{ν_obs}(gT)）になっていない。

**裁定過程（python3 検算）**: 恒等式 g³B_{ν/g}(T)=B_ν(gT) を数値確認（比 1.0000000）。本文の書き方 g³B_ν(T) は、ν=10¹⁸ Hz・T=10⁷ K・g=0.7（鉄輝線・円盤内縁の典型域）で正しい値の **2.70 倍**ずれる。Wien 側では指数関数的にずれが拡大するため、「形を示すだけ」の式としても誤読リスクが大きい。

**採否**: **採用[中]**。番号付きの式として物理的に誤った振動数引数を持つ。修正は1箇所：B_ν(T)·g³ → g³ B_{ν/g}(T)（または等価に B_ν(gT)。Planck 関数のこの性質は一行注記の価値あり）。※フラックスへの変換に必要な 1/D² や立体角の省略は「の形」という但し書きの範囲内と判断し不問。

### 4. SI⇔Gauss 置換方向の記述の混乱（Codex #8）

**指摘**: `appendix/B-units-constants.qmd:225`「Gauss 形では一般に e² → e²/(4πε₀) の置き換えで SI から移れる」― 置換 e²→e²/(4πε₀) は **Gauss→SI** 方向なのに「SI から移れる」と方向が逆に読める。

**裁定過程**: 直下の表で確認：Gauss の α=e²/ħc に e²→e²/(4πε₀) を適用すると SI の e²/(4πε₀ħc) になる（表の全行で同方向）。文は方向を取り違えている（または係り受けが壊れている）。

**採否**: **採用[軽微]**。「Gauss 形の式は e² → e²/(4πε₀) の置き換えで SI 形に移れる」に修正。

## このレビュー依頼の意図

学部後半（B2〜B4）から大学院修士課程（M1）向けの宇宙物理教科書のドラフト v0.1 が完成しました。**正式版（v1.0）に向けて、専門家からの批判的フィードバックを集める** のがこの依頼の目的です。

本書は以下の三つの原則で書かれています：

1. **逆引き**：観測されるスペクトルから出発し、その背後の物理（放射輸送・統計力学・量子論・電磁気・原子物理・QED）を逆向きにたどる
2. **背景物理を曖昧にしない**：各観測量の背後にどの物理がどんな前提のもとで効いているかを章を横断して明示する
3. **天下り的に式を与えない**：プランク分布・リュードベリ公式・振動子強度・Voigt 線形などを結果として与えるのではなく、観測から逆向きに辿って必然性として理解できるようにする

本書の独自性は、(a) 「**逆引き**」を全25章で一貫させたこと、(b) **二つの中心地図**（プランク関数と水素線吸収係数）を全章で再掲し、各章で強調する因子を切り替えてどの物理が今動いているかを示すこと、(c) 最終章で両中心地図が **同じ電磁場-物質相互作用 Hamiltonian の二側面** として QED で統一される構造です。

これらの試みが学生にとって**実際に届くか**を、専門家の目で見ていただきたく思います。

## 見ていただきたい観点（粒度ごと）

ご都合に応じて、以下のいずれかの粒度でお願いします。

### A. 全体構成（短時間で全体感）

- 第I-VIII 部の流れに飛躍はないか
- 「逆引き」が教育的に機能しているか
- 二つの中心地図が学生の頭の中で機能するか
- 第VIII部 QED で連続と線が統合される構造が説得的か

### B. 物理的正しさ（章単位）

- 式・数値・近似の妥当性
- 単位系の混乱の有無（本書は SI 基本、付録 B §B.4 参照）
- 観測値・実例の正確さ
- 過大化・過小化された主張がないか

### C. 教育的観点

- 想定読者レベル（高校〜D、各節に [本文目安：X] と [導出：Y] を記載）が適切か
- 節内「問い」callout が機能しているか
- 章末「使えるようになった道具」が学習目標として有効か
- B2 の学生が読めるか、M1 が物足りないか

### D. 各分野ごとの専門的観点

| 専門分野 | 主に該当する章 |
|---|---|
| 放射輸送・恒星大気 | 第4-5章、第15章、第17章 |
| 統計力学・熱平衡 | 第6-8章、第14章、第16章 |
| 量子論の発生史 | 第9-12章 |
| 電磁気・モード密度 | 第13-14章 |
| CMB・宇宙論 | 第16章、第23章 |
| 高エネルギー天体 | 第18章 |
| 原子・分子分光 | 第19-22章 |
| QED・場の量子論 | 第24-25章 |

ご専門の章を中心に見ていただければ十分です。

## レビュー方法

### 方法 1：ブラウザで読む（最も簡単）

GitHub Pages で公開しています：

→ <https://wadakeiichi.github.io/inverse_text_radiation/>

スマートフォン・タブレットからも読めます。数式は MathJax で表示されます。

### 方法 2：PDF をダウンロード

Zenodo に preprint として登録しています：

→ DOI: 10.5281/zenodo.XXXXXXX （v0.1 登録後にURL記載）

Acrobat 等で注釈を入れて返送いただくと、章・式・段落単位の細かい指摘がいただきやすいです。

### 方法 3：GitHub Issues に直接書く

技術的な指摘や typo は GitHub の Issues に直接投稿いただけます：

→ <https://github.com/wadakeiichi/inverse_text_radiation/issues>

章・節（例：「第18章 §18.2 の不等式 …」）を明記いただけると助かります。

## 2. 新章の骨子

**仮タイトル（採用）**：第19章「自由電子が作る連続光 ― 自由-自由・束縛-自由・再結合」
**ファイル**：`part6/19-continuum-microprocesses.qmd`（ID 接頭辞 `sec-continuum-...`）

> 仮タイトル候補（要確定）：
> 1. **自由電子が作る連続光 ― 自由-自由・束縛-自由・再結合**（採用案。逆引き・現象寄りで §17「…の熱放射」と語調が揃う）
> 2. 連続放射の素過程 ― 自由-自由・束縛-自由・再結合（過程寄り・中立）
> 3. 連続スペクトルはどこから来るか ― 自由-自由・束縛-自由・再結合（§19「線はどこから」と対句）

### 2.1 主題（chapter-overview）

> 第15章で「振動数依存不透明度」を、第17章で電離ガスの自由-自由放射を、いずれも κ_ν・j_ν を**与えられたもの**として使った。本章はその出どころに降りる。希薄プラズマや恒星大気の連続スペクトルを作るのは、**自由-自由**（制動放射）・**束縛-自由**（光電離／再結合連続光）・**束縛-束縛の逆**ではなく自由電子が絡む三素過程である。これらを放出と吸収の両面で、Kirchhoff と Milne の関係、そして Saha 平衡で結び、観測される連続吸収端・電波ターンオーバー・X線指数カットオフ・再結合連続光を逆引きする。最後に、再結合が連続光だけでなく**線スペクトル**（再結合線）も生むことを示し、第VII部への橋とする。

### 2.2 中心地図

既存の連続側中心地図（`_includes/_planck-map-deviations.qmd`）を継承しつつ、本章固有の地図として連続の吸収・放射係数を提示：

$$
\alpha_\nu^{\rm cont} = \underbrace{n_e n_i\,\bar\sigma_{\rm ff}(\nu,T)}_{\text{自由-自由}} \;+\; \underbrace{\textstyle\sum_i n_i\,\sigma_{\rm bf}^{(i)}(\nu)}_{\text{束縛-自由}}, \qquad
S_\nu \overset{\rm LTE}{=} B_\nu(T)\ \text{(Kirchhoff)}
$$

- 占有数 $n_e, n_i, n_X$ は **Saha–Boltzmann** が決める（§19.5）。
- 放出 $j_\nu$ と吸収 $\alpha_\nu$ は **Kirchhoff（LTE）／詳細つり合い（Milne）** で結ばれる。
- 「中心地図の各因子＝どの素過程か」を一枚で見せる方針（既存の中心地図と同じ図法）。

### 2.3 この章で答える問い（callout-question）

- 恒星スペクトルの紫外側の段差（Balmer/Paschen jump）は何の過程か。なぜ「端」が立つのか。
- HII 領域の電波連続光が低振動数で黒体に漸近し、高振動数で平坦になるのはなぜか（§17.6 の微視的根拠）。
- 銀河団の X 線連続光が $e^{-h\nu/k_BT}$ で切れるのはなぜか。
- 星雲の連続光（Balmer 連続光ジャンプ）は何を測らせるのか。
- 同じ再結合が、なぜ連続光（自由-束縛）と輝線（束縛-束縛カスケード）の**両方**を生むのか。

### 2.4 到達目標

- 自由-自由・束縛-自由・再結合の三過程を、エネルギー準位図の上で区別して説明できる。
- 光電離断面積 $\sigma_{\rm bf}(\nu)$ としきい値・連続吸収端の関係を述べられる。
- Milne 関係（詳細つり合い）で $\sigma_{\rm bf}$ と再結合断面積を結び、再結合係数 $\alpha_{\rm rec}(T)$ の意味を説明できる。
- Saha 式を本文の文脈で使い、電離度から連続の $\alpha_\nu, j_\nu$ を自己無撞着に決める道筋を述べられる。
- 連続吸収端・電波ターンオーバー・X線カットオフ・再結合連続光を観測から逆引きできる。

## 4. 演習問題案（5 問＋模範解答）

1. **電波 HII 領域のターンオーバー再訪**：$\alpha_\nu^{\rm ff}\propto\nu^{-2}T^{-3/2}$ から光学的厚→薄の遷移振動数を求め、低振動数で $S_\nu=B_\nu(T_e)\to$ Rayleigh-Jeans $\propto\nu^2$、高振動数で平坦になることを示す。（§17.6 演習 17-5 の微視版・接続）
2. **Balmer 連続光ジャンプ**：束縛-自由しきい値 $\lambda=3646$ Å での吸収係数の不連続を $\sigma_{\rm bf}$ から見積もり、恒星大気（吸収端）と星雲（再結合放出端）で符号が逆に見える理由を述べる。
3. **Milne 関係**：詳細つり合いから $\sigma_{\rm bf}(\nu)$ と再結合断面積 $\sigma_{\rm rec}(v)$ の比を導き、再結合係数 $\alpha_{\rm rec}(T)=\langle\sigma_{\rm rec}v\rangle$ の温度依存（$\sim T^{-1/2}$ 級）を確かめる。
4. **X 線制動放射で銀河団温度**：$F_\nu\propto e^{-h\nu/k_BT}$ の指数カットオフから $T_e\sim10^7$–$10^8$ K を読む。（§18 と接続）
5. **H⁻ のしきい値**：束縛エネルギー 0.754 eV から bf しきい値波長（≈1.64 μm）を求め、なぜ太陽光球の可視連続不透明度を中性 H の bf 端（紫外）ではなく微量の H⁻ が担うのかを、Saha（自由電子供給）と絡めて説明する。

**残論点**

4. **H⁻ 連続吸収（(b) 踏み込む＝決定）**：H⁻ は太陽型・低温星（G・K型）の可視〜近赤外で連続不透明度を支配する主役。§19.3 で**踏み込んで**扱う：
   - H⁻ の **bound-free**（束縛エネルギー 0.754 eV → しきい値 約 1.64 μm、可視〜近赤外をカバー）と **free-free**（より長波長側）の両寄与。
   - 中性水素の bf 端は紫外なのに、可視で太陽光球が不透明な理由が H⁻ であること（中性 H 主成分なのに連続不透明度は微量イオンが担うという逆引き的な妙）。
   - **低温星で卓越する条件**：自由電子の供給源は金属（低電離ポテンシャル元素）の電離、豊富な中性 H が H⁻ を作る。よって H⁻ 不透明度は電子密度（≒金属量・Saha）に依存 → §19.5 の Saha と接続。高温星では H が電離して H⁻ が消え、bf/ff（H, He）と電子散乱に主役交代。
   - 演習に H⁻ しきい値波長（0.754 eV→1.64 μm）の計算を 1 問追加（演習を 5 問に）。
   - 図：§19.3 の連続吸収端図に H⁻ の寄与曲線（可視に山）を重ねる変種を検討。
